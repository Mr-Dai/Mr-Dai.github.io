vector属于C++中STL的一员，是一个可自动改变大小的动态数组，在空间上由若干个连续的线性空间组成，借由动态地申请或释放线性空间以动态地改变自身容量大小。正式由于这种情况，vector每次的添加或删除元素都有可能导致元素的空间分布发生巨大改变，因此每次insert或erase时，作为参数用于指向目标元素的迭代器都将在操作完毕后被废除，无法再被使用，取而代之的将是insert与erase函数返回的新迭代器。


vector常用函数如下：


构造函数

  • vector()：默认构造函数，构造一个空vector
  • vector(int nSize)：构造一个vector，其大小为nSize,内部的元素将调用其自身的默认构造函数进行初始化
  • vector(int nSize,const T& t)：构造一个vector，其大小为nSize，所有元素将被置为t
  • vector(const vector&): 拷贝构造函数（深复制）
  • vector(begin, end)：将begin与end之间包含的元素复制到新的vector中（深复制）

增加函数

  • void push_back(const T& x)：将元素x添加至vector的末尾
  • iterator insert(iterator it,const T& x)：在迭代器it指向的位置插入新元素x，原本的元素将被后移至新元素之后， 返回指向插入后新元素所在位置的迭代器。注意：在执行过后，迭代器it将无法再继续使用
  • iterator insert(iterator it,int n,const T& x)：在迭代器it指向的位置插入n个新元素x，返回指向插入后首个元素所在位置的迭代器
  • iterator insert(iterator it,const_iterator first,const_iterator last)：在迭代器it指向的位置插入first与last迭代器之间的所有元素

删除函数

  • iterator erase(iterator it)：删除迭代器it指向的元素，后续元素前移，返回指向被删除元素下一个元素的迭代器。注意：在执行过后，迭代器it将无法再继续使用
  • iterator erase(iterator first, iterator last)：删除向量中[first, last)中的元素
  • void pop_back()：删除向量中最后一个元素
  • void clear()：清空向量中所有元素

遍历函数

  • reference at(int pos):返回pos位置元素的引用
  • reference front():返回首元素的引用
  • reference back():返回尾元素的引用
  • iterator begin():返回向量头指针，指向第一个元素
  • iterator end():返回向量尾指针，指向向量最后一个元素的下一个位置
  • reverse_iterator rbegin():反向迭代器，指向最后一个元素
  • reverse_iterator rend():反向迭代器，指向第一个元素之前的位置

判断函数

  • bool empty() const:判断向量是否为空

大小函数

  • int size() const:返回向量中元素的个数
  • int capacity() const:返回当前向量张红所能容纳的最大元素值
  • int max_size() const:返回最大可允许的vector元素数量值

其他函数

  • void swap(vector&):交换两个同类型向量的数据（交换内部存储的数列指针）
  • void assign(int n,const T& x):设置向量中第n个元素的值为x
  • void assign(const_iterator first,const_iterator last):向量中[first,last)中元素设置成当前向量元素（深复制）
