二维矩阵降维

可利用reshape函数

    matrix(1:3,1:3) = 1
    matrix = reshape(matrix', 1, [])

二维矩阵排序

直接对二维矩阵使用sort函数只能逐行或逐列排序

解决方案：先用reshape函数降维，再用sort函数排序

    matrix = [ 1,2,3; 4,5,6; 7,8,9 ]
    matrix = reshape(matrix', 1, [])
    matrix = sort(matrix)
